# Financial Advisory Report (FAR) — Web Form

A web-based fact find and needs analysis tool for financial advisers.

## What it does
- Collects client personal information, income, expenses, assets, liabilities, insurance coverage, and planning goals
- Calculates financial ratios, insurance gaps, and retirement shortfall
- Generates a summary advisory report instantly in the browser
- No data is stored or transmitted — everything runs locally in the browser

## How to use
👉 [Open the FAR Form](https://Jessica-mtm.github.io/index.html/)

## Sections
1. Personal Information
2. Income & Expenditure
3. Assets & Liabilities
4. Insurance Coverage
5. Planning Goals & Needs
6. Planning Assumptions
7. Generated Report

## Notes
- All calculations are estimates for planning purposes only
- No client data is saved — each session starts fresh
- Use Print / Export to save a copy of the completed report
